# 📊 Encuesta Online
Este es un proyecto desarrollado para gestionar encuestas en tiempo real. 

## 🚀 Características
* Interfaz dinámica y responsiva.
* Validación de datos de usuario.
* Almacenamiento de respuestas en base de datos.

## 🛠️ Tecnologías utilizadas
* HTML5 / CSS3 / JavaScript
* [Aquí agrega si usas React, Node, Python, etc.]

## 📦 Instalación y Uso
1. Clona el repositorio:
   `git clone https://github.com/majuanjose/encuestaonline.git`
2. Abre el archivo `index.html` en tu navegador o ejecuta `npm start`.
